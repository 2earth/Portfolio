User-Centred Design Project: Inclusive Self-Checkout
Team of 2 | 3 weeks | 2017

I took part in a two-week design project supported by disability charity Designability to identify and propose improvements to supermarket self-checkout machines, making them more accessible for stroke survivors. This began by interviewing a man who had suffered a stroke two years prior, and brainstorming the issues he faced using self checkouts.
The aspects he struggled with were then summarised, allowing relevant concepts to be generated.
Concepts were then evaluated using Multi-Criteria Decision analysis against Designability's criteria.
The concept that was pursued set out to simplify the structure of the payment task by carrying out all interactions in one location, and to be operable with one hand. This was explored in detail.
This led to the creation of numerous quick cardboard prototypes, which were tested by the user, providing a final design direction.
The pivoting handle was a key component and was modelled using Autodesk Fusion 360, taking into account ergonomic data to ensure an inclusive sizing.
The final prototype was then created using rapid prototyping.
A storyboard shows the method with which the user would interact with the design.
(1) The product would allow the user to place his wallet on the table

(2) This would aid his ability to gather, pick and place coins

(3) The user could slide coins into the machine one at a time or lift the handle, which requires less dexterity than a sideways slot.

(4) The machine would process and return the correct change to the same tray, simplifying the number of places the user needs to pay attention to.

(5-6) The user could then gather all his change into the palm of his hand in one motion, rather than picking individual coins with one hand.
