Electronics/Arduino
I've been using the Arduino Uno for hobby projects since before University, but have since applied my skills to more serious projects.
For a mechatronic design project I created, tested and debugged the electronics for the design. I designed and tested each sub-system independently on a breadboard, then soldered a shield with all circuits integrated. Debugging used continuity tests and oscilloscope probing to identify soldering mistakes.
As a tangible demonstration, I created a prototype centrifuge for a reverse engineering and disruptive innovation project. This was built entirely on a breadboard and featured button inputs to control a screen, an interlock and warning LED.
