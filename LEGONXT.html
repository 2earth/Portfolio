LEGO NXT Maze Solver
The task was to program a LEGO NXT vehicle to solve a wooden maze using the left-hand wall-following algorithm, then return to the start point via the shortest available path. All navigation was made by following black lines and using ultrasonic distance sensors to detect the position of walls.
The most challenging programming challenge was creating an algorithm which optimised the robot's path.
Once the algorithm was planned, this was written into the graphical language LabView
