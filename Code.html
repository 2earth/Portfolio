MATLAB
My most confident language, I've used MATLAB extensively during university.
My largest MATLAB project was a mechatronics design project in which I wrote all the code. The hardware would scan a rectangular area for magnets, then detect and optimise a path to seek each one.
The raw data collected by moving the rig and taking voltage readings from a hall-effect sensor needed to be processed to return reliable data.
When moving the gantry up or down a lightgate was used to detect stalls, preventing misalignment. When rotating the gears a continuous rotary potentiometer was used to operate the DC motor in closed-loop feedback, allowing accurate and precise position control.
In my second year I worked on an individual project to model the heat transfer through an insulating tile on a space shuttle during re-entry. This required the integration of heat transfer and modelling techniques theory with practical coding skills.
The final code modelled the heat transfer across time and through the tile's thickness, allowing the user to alter parameters within a graphical user interface (GUI) to observe effects. It also allowed the user to set a maximum inner surface temperature and use the shooting method to identify the required thickness of tile.
In my first year I created a program which modelled the trajectory of a 'parcel' fired at a certain angle and speed, and with a parachute being deployed to reduce the surface impact speed. Once more, the shooting method was used to calculate the initial conditions required to impact a target.
In my first year I conducted a practical laboratory experiment, collecting data from a mountain bike spring-damper suspension system. This data was plotted in MATLAB to support theoretical conclusions.

C++
In my free time I have created a number of small Arduino projects, all of which have been programmed using C++. An example of this is a prototype laboritory centrifuge I built and programmed in two days.

the centrifuge had a microswitch used as an interlock, which would prevent the shaft turning if the lid was open. Once the lid was shut the user could increase or decrease the rotational speed. All of this information was displayed on a screen.

Labview
I used the graphical programming language Labview to control a LEGO NXT which solved a maze autonomously.
This involved using familiar coding structures such as if and while loops, as well as boolean and logical tests. Shown above is the code to ensure the robot followed a black line.
The unique part of the code was as the robot navigated the maze it stored a record of movements, encoded by an absolute "compass".
